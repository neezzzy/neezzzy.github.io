# Portfolio Website

Personal portfolio for Russell Nizamov, positioned around business systems, automation, and AI workflow development.

Live site: https://neezzzy.github.io/

## Focus

This portfolio highlights practical projects that solve clear workflow problems: local-first tools, dashboards, education products, rule-based workflow MVPs, and automation-friendly interfaces.

The site now includes a recruiter-focused snapshot for business systems roles and a real-world WordPress setup/launch project for Definitive Jiu Jitsu.
